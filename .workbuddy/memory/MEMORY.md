# ShrikeDxf 项目记忆

## 用户偏好

- **修改前预览确认**：任何代码/文件修改前，先把拟改动的内容以文本形式输出，等用户确认后再执行修改。

## 项目信息

- 项目类型：C++ / Qt，DXF CAD 文件解析与绘制
- 主要目录：`ShrikeDxf/Manager/`（控制器、数据管理、交互）、`libdxfrw/`（DXF 读写库）、`EntitiesLib/`（实体定义）、`dxflib/`（DXF 库）

## 命名规范

- C++ 成员变量：`m_camelCase`（保留 `m_` 前缀，去掉匈牙利类型前缀如 `e`/`pt`/`vec`/`p`）
- C++ 函数：首单词小写小驼峰
- 构造函数参数：纯小驼峰，不带 `p` 指针前缀
